\doxysubsection{Detailed Description}
Compress/decompress data using the huffman coding technique.

\mbox{\hyperlink{classhuffman_1_1HuffmanCoding}{Huffman\+Coding}} implements the algorithms described in the \href{https://en.wikipedia.org/wiki/Huffman_coding}{\texttt{ Huffman Coding}} Wikipedia page.
